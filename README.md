# Social Media — Effects
A website built in 2 hours, in Notepad++, during an inter-collegiate web design competition at DHB Soni College. No VSCode. No shortcuts. Won.

---

## The Story
The theme was given on the spot: Social Media and its effects on student life. While most participants focused on getting content on the page, the approach here was different: get the structure right, then make it look like someone actually cared.
Turns out, that was enough. Only a handful of people implemented CSS at all. This was the one that did it properly.

## What's on the page
- Sticky navbar with smooth anchor navigation
- Hero section with a glitch text-shadow effect on hover
- About section: two-column layout with image
- Effects section: pros and cons side by side
- Contact / Feedback form (UI only)
- Footer

## Design choices worth noting
- The entire site runs on a black background with white text and translateY + box-shadow hover effects on every interactive element. Repetitive by design — consistency is the polish.
- The hero text uses offset red and blue text-shadows on hover to create a glitch effect. Simple CSS, high impact. That one line probably won the competition.

## Constraints
- Built in Notepad++, VSCode was not allowed
- Completed in 2 hours
- No frameworks, no libraries. Pure HTML and CSS

## Takeaways
- Simple, repetitive design is underrated. Every section follows the same visual language: same border-radius, same hover transitions, same spacing rhythm.
- It doesn't need to be complex to look complete.
Don't add elements to fill space. Every section on this page earns its place.
